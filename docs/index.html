Obsidian Plugin Collection

The plugins I run in my Obsidian vault, bundled as ready-to-install zips. Each entry links to the plugin's Obsidian community listing and its source repository.

Download from Releases View Repository
Install Extract a plugin zip (or all-plugins.zip) into your vault's .obsidian/plugins/ directory, reload Obsidian, then enable the plugin under Settings > Community plugins. Zips contain only main.js, manifest.json, and styles.css — no personal settings.
